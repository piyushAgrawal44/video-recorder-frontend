# 📽️ video-rev

A modern React application using Vite, TypeScript, Tailwind CSS, and Socket.IO.

---

## 🚀 Project Setup

### 📦 Prerequisites

- **Node.js** ≥ 18.20.4

---

## 🛠️ Installation & Setup
1. npm install
2. create .env file in root directory and copy content of  env.example
3. npm run dev

## Build 
1. npm run build
